 نبذة عن المشروع
متجر عطاء تهامة هو متجر إلكتروني احترافي متعدد الأقسام يهدف إلى تقديم تجربة تسوق سهلة وآمنة للمستخدمين، مع واجهة عربية حديثة وتصميم متجاوب يعمل على جميع الأجهزة.

يتيح المتجر للعملاء:
- تصفح المنتجات حسب الفئات
- البحث المتقدم عن المنتجات
- إضافة المنتجات إلى السلة
- إتمام الطلب والدفع
- إدارة الحساب الشخصي
- تتبع الطلبات
- التواصل مع خدمة العملاء
